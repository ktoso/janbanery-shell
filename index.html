janbanery-shell by ktoso

A Kanbanery.com command line client. With FULL API coverage!

Install

Download the latest executable jar from: https://github.com/ktoso/janbanery-shell/blob/master/release/uber-janbanery-shell-1.0-SNAPSHOT.jar and just run it using: ```bash java -jar uber-janbanery-shell-1.0-SNAPSHOT.jar ```
